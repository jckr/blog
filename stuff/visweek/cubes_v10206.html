(well this will be made more elaborate later but for the time being...)

rotate cube with arrows.

turn cube into square pie with space bar.

when cubes are arranged in square pie, change variable order with S sort by variable order <-> sort by value.

filtering: click on variable to see possible values, click on value to exclude this value, double click to exclude all others.

if one value is excluded for one variable, it's possible to clear the filtering for this variable by clicking on the green flag.

values filtered out have a darker background.

faces: click on a variable to assign this to the face of the cube facing you. Variables already assigned to a face have a darker background. I'll make this more visual soon.

details: click on a cube for more information.
